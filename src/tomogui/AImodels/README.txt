Place AI model weight files (.pt / .pth) here.

Expected default path used by tomogui:
    AImodels/datav2_518_full_finetune/epoch_10.pt

The path is relative to this directory (src/tomogui/AImodels/).
The GUI's "AI Model" field defaults to this path on startup and can be
overridden per-run via the Browse button.

Model files are typically large (> 100 MB) and are not stored in git.
Download them separately and place them under this folder.
