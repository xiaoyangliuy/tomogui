Batch Processing Guide
======================

Complete guide to batch processing multiple datasets efficiently.

See :doc:`../features/batch_tab` for detailed interface documentation.

Overview
--------

Batch processing allows you to:

- Process multiple datasets automatically
- Utilize multiple GPUs in parallel
- Manage COR values for each file
- Execute on local or remote machines
- Monitor progress in real-time

Quick Start
-----------

Basic batch workflow:

1. Navigate to Batch Processing tab
2. Click "Refresh File List"
3. Enter COR values for files
4. Select files to process
5. Configure machine and GPUs
6. Click "Run Try on Selected" or "Run Full on Selected"

See :doc:`../features/batch_tab` for complete details on all features.
